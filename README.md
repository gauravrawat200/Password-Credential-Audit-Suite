# Password Cracking & Credential Attack Suite

## Project Overview

This project is developed for educational and cybersecurity auditing purposes. It demonstrates how password security can be analyzed using various techniques such as dictionary generation, hash analysis, brute-force simulation, and password strength evaluation.

## Objectives

- Generate custom password dictionaries
- Analyze password hashes
- Simulate dictionary-based password testing
- Estimate brute-force resistance
- Evaluate password strength
- Generate security audit reports

## Tools Used

- Python 3
- Kali Linux
- hashlib
- passlib
- GitHub

## Project Modules

### 1. Dictionary Generator
Creates custom password wordlists using names, dates, common patterns, and mutations.

### 2. Hash Analyzer
Identifies common hash algorithms such as:
- MD5
- SHA1
- SHA256
- SHA512

### 3. Dictionary Attack Simulation
Demonstrates password testing against user-created sample hashes in a controlled environment.

### 4. Brute Force Estimator
Calculates password search space and estimated cracking time.

### 5. Password Strength Analyzer
Checks:
- Length
- Complexity
- Entropy
- Predictability

### 6. Report Generator
Creates a final audit report containing findings and recommendations.
